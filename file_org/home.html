File System Simulator

Welcome to the File System Simulator! This interactive tool allows you to practice and understand file system operations in a visual environment. Create directories, navigate between them, and add files using the command buttons.

Start Simulation

Create Directory Structure

Use the mkdir button to create directories and organize your virtual file system.

Navigate Directories

Change directories with the cd buttons, moving up and down your directory tree.

Create Files

Add files to your directories using the create button to build a complete file structure.
