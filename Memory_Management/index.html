Memory Management Techniques

MFT (Multiprogramming with Fixed Tasks)

Memory is divided into fixed-size blocks. Each process gets a block. If the process is smaller than the block, space is wasted (internal fragmentation).

MVT (Multiprogramming with Variable Tasks)

Memory is allocated dynamically to processes. It reduces internal fragmentation but may cause external fragmentation over time.

Demand Paging (Virtual Memory)

Processes are divided into pages and only required pages are loaded into memory frames when needed. Reduces memory usage and allows multitasking efficiently.
