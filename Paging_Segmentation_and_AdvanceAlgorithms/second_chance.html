The Second Chance (or Clock) algorithm is an enhanced version of FIFO (First-In-First-Out). It gives each page a "second chance" before replacing it by checking its reference bit.

üí° **How It Works:** When a page is about to be replaced, if its reference bit is 1, it's not removed immediately. Instead, the bit is cleared (set to 0) and the algorithm gives it another round in memory. Only pages with reference bit 0 are replaced.

üß† This way, frequently used pages stay in memory longer, improving efficiency and reducing page faults.
